# InstantSaver

InstantSaver is a free tool to preview and download Instagram Reels, Posts, Carousels, and YouTube Videos/Shorts with one click.  
Built using React (frontend) and Node.js + Express + yt-dlp (backend).

## Features
- Instagram Reels & Posts downloader
- YouTube videos & Shorts downloader
- Fast preview before download
- Node.js backend with yt-dlp merging video + audio
- Clean and responsive UI

## Tech Stack
- React
- Node.js / Express
- yt-dlp
- Netlify (frontend hosting)
- Render / Railway (backend hosting)
